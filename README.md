Private Extension Methods (2026 version)
==============

This is a rewrite of an earlier proposal for private extension methods.

For the current draft go here:
* https://github.com/aroffringa/stdcxx-privext-2026/blob/main/proposal/draft.md

This idea was discussed on the standard proposal e-mail list with the start of
the thread here:
* https://lists.isocpp.org/std-proposals/2026/04/18185.php

The original (2013) proposal can be found here: https://github.com/mateofio/stdcxx-privext

The 2013 proposal has been submitted to the ISO C++ committee:
* http://open-std.org/jtc1/sc22/wg21/docs/papers/2014/n3863.html
